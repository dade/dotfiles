# 💤 LazyVim Configuration

These are my personal configs for LazyVim, with some extra plugins to achieve what I need from neovim.

## LazyGit
Adding LazyGit just so that I don't need to open a new terminal in case I've opened neovim without first initializing tmux.

## godotdev
This is the nvim part of the setup for godotdev. There are other scripts I'm running on the system level that handles the remote aspect of using nvim as my IDE. This is all explained on the [godotdev.nvim readme](https://github.com/Mathijs-Bakker/godotdev.nvim?tab=readme-ov-file#installation-lazynvim)

## undotree
Because this plugin is legitimately awesome.

## Render Markdown
Makes life a little nicer when writing readmes, working with obsidian files in nvim, etc.
